### 设计模式
#### 1.目的
本项目主要为设计模式的学习资源源码。
#### 2.使用
git clone git@github.com:lightClouds917/designMode.git
拉取项目到本地，不同的设计模式单独存储在对应的包下，部分设计模式演示或者验证时，是在其同包名下的测试类中进行。
#### 3.更新进度
##### 单例模式
singleton包下，类中有注释，有以下5种：
- 饿汉式
- 懒汉式
- 加锁双重检查
- 内部静态类
- 枚举
##### 代理模式
proxy包下，类中有注释，有以下3种：
- 静态代理
- jdk动态代理
- cglib动态代理

##### 简单工厂模式
factory包下

##### 建造者模式
builder包下

##### 观察者模式
observer包下
